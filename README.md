# Twich chat battle

## About
Simple chat game, where people type into chat. Number of worlds indikates damage that is dealt to monster. If monster died, new monster is created.
If Twitch chat died, all game stats are reseted.

## Requirements

* `node` **>=** `10.13.0`
* `npm`

# Setup

## Installation

```sh 
$ npm ci
```

## Development

```sh 
$ npm run serve
```
